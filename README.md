## 🎯 Featured Projects

### 🚨 [Protocol One](https://github.com/varadb2006/protocol-one)
An LMS-style portal for disaster management education. Built as a solution to the problem statement provided by NDMA (National Disaster Management Authority) during a hackathon.
- **Tech:** React, Flask, SQL
- **Features:** Course modules, Interactive learning modules, Disaster preparedness training
- **Impact:** Empowering users with critical disaster management knowledge

### 🏘️ [Vicinity](https://github.com/varadb2006/vicinity)
A location-based community platform bringing neighbors together.
- **Tech:** React, Django, SQL
- **Features:** Real-time updates, User profiles, Community features

### 🌱 [Calyx Smart Irrigation](https://github.com/varadb2006/calyx-smart-irrigation)
An intelligent irrigation system using IoT and machine learning for optimal water management.
- **Tech:** Python, Flask, SQL, ML algorithms
- **Impact:** Sustainable agriculture, Water conservation
- **Key Feature:** AI-driven watering schedules based on soil moisture & weather

### 🐄 [Bharat Pashudhan](https://github.com/varadb2006/bharat-pashudhan) *(In Progress)*
A comprehensive livestock management platform for Indian farmers.
- **Tech:** MERN Stack, AI/ML
- **Goal:** Empowering farmers with digital livestock management tools
- **Status:** 🚀 Active Development
